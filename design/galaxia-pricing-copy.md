## 6. What must ship before we can charge $9.99

Verified gaps against the competitor we're priced under. CHANI's charts include the lunar nodes, MC, IC, DC, Chiron, and Black Moon Lilith. Ours have none of them.

1. **Chiron, North & South Node, Lilith, MC/IC/DC.** An astrology-literate trial user checks for these in the first ten minutes.
2. **Compare guidance must be per-person.** It currently returns word-for-word identical text for two different charts. This is the feature the price rests on.
3. **A forward transit view.** One daily line is not a reason to open the app on day 9.
4. **Saved readings / exportable charts.** Accumulated value is what makes leaving feel like a loss.

Ship as-is and the trial user finds the gaps before the fourteenth day.
